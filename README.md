# Base Structure

## Battle UI

- blue sx
- green dx
- lifebars, names
- command zone

## Combat

- turn order
- damage calculation
- state (lifebars, turns, animations on atk, animations on hit with "shake")

## Flow

- loading
- start anim
- choose atk -> anim -> calc -> end anim -> state (pv, anims)
- enemy turn
- repeat until either lifebar reaches 0
- show win/defeat screen and message
